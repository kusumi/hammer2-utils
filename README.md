hammer2-utils ([v0.4.0](https://github.com/kusumi/hammer2-utils/releases/tag/v0.4.0))
========

## About

HAMMER2 userspace utilities in Rust

## Supported platforms

Linux / FreeBSD

## Requirements

Rust 1.86.0 or newer

## Build

    $ make

## Install

    $ make install

## Uninstall

    $ make uninstall
