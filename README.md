hammer2-utils ([v0.2.1](https://github.com/kusumi/hammer2-utils/releases/tag/v0.2.1))
========

## About

HAMMER2 userspace utilities in Rust

## Supported platforms

Linux / FreeBSD

## Requirements

Rust 1.81.0 or newer

## Build

    $ make

## Install

    $ make install

## Uninstall

    $ make uninstall
